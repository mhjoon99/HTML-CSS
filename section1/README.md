## 섹션 1. HTML Basics

### &lt;html lang=“en”&gt;
이 문서가 html로 작성됐음을 알림.
html 문서의 시작과 끝을 알리는 태그. lang=문서의 언어

### &lt;head&gt;
- 메타데이터와 타이틀을 넣어주는 곳.
>> - 메타 데이터 : 웹 페이지에서 직접적으로 보이지는 않지만 페이지 인코딩 방식이나 검색엔진 등 많은 설정 할 수 있음.
>> - 타이틀 : 브라우저 탭에 나오는 제목

### &lt;body&gt;
- 주로 화면에 보이는 것들이 들어감.
- HTML 문서에서 단 하나의 <body>만 존재
